# RAG OpenAI Native

A lab project for implementing Retrieval-Augmented Generation (RAG) using OpenAI's native capabilities.

## Overview

This project demonstrates how to build a RAG system leveraging OpenAI's API and native features.

## Setup

1. Configure your environment variables in `.env`
2. Install required dependencies
3. Run the notebook: `rag_openai_native.ipynb`

## Project Structure

- `rag_openai_native.ipynb` - Main notebook containing the RAG implementation
- `data/` - Directory for storing data files and documents
- `lab_summary.md` - Summary of lab findings and results

## Requirements

- Python 3.9+
- OpenAI API key
